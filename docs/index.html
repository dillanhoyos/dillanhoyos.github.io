    Games and Implementation

    Pre Mortal VR

    In the immersive world of this VR game, I proudly assumed the role of the Lead Sound Designer. Within this capacity, my diverse range of responsibilities encompassed the creation of audio assets and the development of cutting-edge audio runtime systems tailored specifically for the VR interactions. All of these endeavors were executed using the powerful Unity engine. Steam Page Premortal VR

    Dream Machine

    The Dream Machine Project is a multi-player, VR, interactive platform that provides a unique stage for artists to showcase their music and educate players about their musical influences in an engaging and entertaining manner. In this project, I served as a Unity Developer and Technical Audio Designer. My responsibilities included the development and implementation of various technical aspects of the project using Unity game engine. Additionally, I was responsible for the technical design and implementation of audio for the project. The technologies used in this project included NormCore and Wwise, which served as the middleware for audio management and integration. These technologies were instrumental in delivering a seamless and immersive experience for the users and ensured that the audio design was of the highest quality.
