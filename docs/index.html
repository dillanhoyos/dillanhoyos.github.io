    Pre Mortal VR

    In the immersive world of this VR game, I proudly assumed the role of the Lead Sound Designer. Within this capacity, my diverse range of responsibilities encompassed the creation of audio assets and the development of cutting-edge audio runtime systems tailored specifically for the VR interactions. All of these endeavors were executed using the powerful Unity engine.

    Unreal_Wwise Implementation Gameplay

    This gameplay video of the game I designed features intuitive level design and immersive sound created with Wwise, Blueprint, and C++. Traverse challenging levels while being fully immersed in a soundscape that matches the intensity and tone of the action, creating a truly engaging experience.

    Unity_Wwise Implementation Gameplay

    Experience immersive gameplay in this level with expertly designed sound using Wwise, C#, and Unity. The use of C# programming language ensures a seamless and dynamic gameplay experience, while Wwise takes the sound design to the next level, creating an unforgettable and fully immersive game world.

    Twin-Edge

    Music Implementation using C# and Unity. Utilizing Unity game engine, I was responsible for ensuring that the audio aspects of the game were implemented to the highest technical standards. My role as a musician involved composing and incorporating a fitting musical score that complemented the rhythm-based gameplay.
